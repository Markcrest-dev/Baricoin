# Baricoin Mobile App 📱

A cross-platform mobile application for the Baricoin platform, powered by Expo and React Native.

## 🛠️ Tech Stack
- **Framework**: React Native / Expo
- **Navigation**: Expo Router
- **Styling**: NativeWind (Tailwind CSS for Mobile)
- **State Management**: Zustand
- **Icons**: Lucide React Native

## ⚙️ Installation

1. Install dependencies:
   ```bash
   npm install
   ```

2. Install Expo Go on your mobile device (iOS/Android).

## 🏃 Running the App

```bash
# Start the Expo development server
npm run start
```

Once the server is running, you can:
- Scan the QR code with your phone (Expo Go app).
- Press `a` for Android Emulator.
- Press `i` for iOS Simulator.
- Press `w` for Web Version.

## 📁 Project Structure
- `app/`: Expo Router file-based navigation.
- `components/`: Mobile-specific UI components.
- `services/`: API integration and logic.
- `store/`: state management.
- `assets/`: Images, fonts, and splash screens.
